## Component Description
To add a description to a component you can add a `description` key to the `createMainStory` function. You can pass `String`, `HTML`, `React Component` or use the provided `Description` component like the example below.

```javascript
import { Description } from '../utils';
```

## Story Description
A description above a sub-story.

Descriptions can be `Strings`, `Marrkdown` or `MDX`.
<img style={{ width: '80%' }} src={storyDescription} alt={'story description'}/>

### Description with `generateStory`
To add a description above a sub-story that was generated by `generateStory`, you can use the `description` property of the parameter object of the `generateStory` function.

<Source language={'javascript'} dark code={storyDescCode} />

### Description with `generateSubstories`
To add a description above a sub-story that was generated by `generateSubstories`, you can use the `description` property of the parameter object of each sub-story of the `generateSubstories` function.

<Source language={'javascript'} dark code={storiesDescCode} />
