# Installation
<br />
<br />

## 1. Create Repository

To start using the `Component Library Template` create a repository on `GitHub` with this [template](https://github.com/Blue-Functor/component-library-template).

## 2. Clone the new repository

Clone the new repository using:
```
$ git clone <YOUR_REPO_URL>
```

## 3. Install dependencies

To install development dependencies use:
```
$ yarn install
```
or
```
$ npm install
```

## 4. Start Storybook

To start the Storybook environment run:
```
$ yarn start
```
or
```
$ npm start
```

> Note: this command will start Storybook with `--no-manager-cache`. If you wish to run the `default` Storybook command use:
> ```
$ yarn storybook
```
or
```
$ npm storybook
```

Then navigate to [http://localhost:6006](http://localhost:6006) where your new component library is running.

## Troubleshooting
> For troubleshooting please refer to the original [Storybook Guide](https://storybook.js.org/docs/react/get-started/introduction), so far we are only using their default configuration.
